Leitura de Frações

Vamos estudar as partes de uma fração a partir de uma pizza?





Na imagem acima temos uma pizza, imagine que ela tem 300 gramas, e que 3 pessoas irão comê-la, quantas gramas de pizza cada um vai comer??










Exercício 1

Responda:


1. Dividindo a barra de chocolate entre 2 pessoas, quantas gramas de chocolate cada uma vai receber?


2. Dividindo a barra de chocolate entre 4 pessoas, quantas gramas de chocolate cada uma vai receber?





Após responder clique nas respostas para vê-las


Resposta 1

100 gramas para cada pessoa.


Resposta 2

50 gramas para cada pessoa.
